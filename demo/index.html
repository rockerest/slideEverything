Welcome to the demo & documentation page for slideEverything. I'm excited you're here!

I don't want to waste your time, so let's get right into it!

The HTML

The HTML that this plugin requires is very simple. First, you need to define a window that the slider will inhabit.

<div id="slider-a" style="width: 800px; height: 500px;">
</div>

Then, drop everything you want inside that slider. Keep in mind that each element that is a direct descendant of the outer slider will be considered a slide, so keep things well grouped!

Make sure you explicitly define the width and height of your slider. The plugin relies on a defined size to create the slides, so it's behavior could be erratic if you don't.

<div id="slider-a" style="width: 800px; height: 500px;">
    <p>A String</p>
    <div>
        <p>A String in a Slide</p>
        <p>A String in the same Slide</p>
    </div>
    <img alt="An Image" src="http://neilvn.com/tangents/images/software/neatimage/neat-image-off-lrg.jpg" />
</div>

The Javascript

First of all, you need to use jQuery. After all, this is a jQuery Plugin.

<script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.7.1/jquery.min.js"></script>

Once you've got jQuery loaded up, pull in the plugin.

<script type="text/javascript" src="jquery.fn.slideEverything.js"></script>

Finally, you're going to need to do a little jQuery coding yourself. Pop open a new .js file (I named mine index.js to make things easy), and try this code for starters.

$(function()
    {
        $('#slider-a').slideEverything();
    }
);

Once you've got your custom javascript, you'll need to include it in your page, so drop a third script tag in there after the plugin.

<script type="text/javascript" src="index.js"></script>

Advanced Options

The goal of this plugin is to give a ton of control of the slider - in a simple way - to the user, whether that user is the web developer, or the intended target using the web site.

This plugin is in it's absolute infancy, so much of that functionality isn't yet present, but you have a bunch of control over how it currently works.

Methods:

$().slideEverything('init'); is identical to calling $().slideEverything();.

$().slideEverything('start'); will start the slideshow automatically playing.

$().slideEverything('stop'); will stop the slideshow if it is automatically sliding.

$().slideEverything('move'); will progress the slider/slideshow to the next queued slide.
The next queued slide is determined by the options object (more on that later).

$().slideEverything('update', options); will load a new set of options into the slider (more on the options object later).

$().slideEverything('destroy'); will completely remove the slider/slideshow from memory. It should stop running AND EXISTING.

$().slideEverything('debug'); will dump the options object (more on the options object later) to the console.

To control how the plugin works, you can initialize it ($().slideEverything(options);) or change it's operation on the fly ($().slideEverything('update', options);) using an options object.

The object is normal JSON and looks just like any other jQuery object.

Not a lot is exposed right now, since the plugin doesn't currently implement many of it's goals, but the options listed below are available for use.

Options:

{"pre" : "[string]"} is the prepended string for inserted DOM elements.
Unless you are encountering namespacing overlap, you should probably leave this one alone.
default: "SE_"

{"defer" : [true|false]} determines whether the slider should automatically begin playing.
Remember that the property is defer, so false begins the slideshow automatically, while true does not start automatically.
default: false

{"delay" : [number]} determines the time in milliseconds before a new slide is brought in automatically.
Only takes effect if "defer" : false.
default: 3500

{"easing" : "[linear|swing]"} determines the method that the new slide will enter the viewer.
The only options available by default in the jQuery library are "linear" and "swing."
default: "swing"

{"timing" : [number]} determines the time in milliseconds that it takes a new slide to complete it's animation.
default: 1000

{"animation" : "[push|over]"} determines which slide animation to use.
Current options are "push" - which pushes the old slide out at the same rate the new slide comes in - and "over" - which slides the new slide in over the old one.
default: "push"

Advanced Demo

Using the advanced options, we change the operation of the slideshow we created above.

Open up your custom jQuery file that you wrote earlier, and change it to look like this:

$(function()
    {
        $('#slider-a').slideEverything(
            {
                "animation" : "over",
                "easing" : "linear"
            }
        );
    }
);
