About
=====
I got tired of sliders that did one thing, and locked users into that one thing.  
I wanted a slider that would do everything, and give the user the power to control how it works.  

Roadmap
=======
Directional Sliding  
	-right  
	-down  
	-up  
  
UI for manual control
